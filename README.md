## 📸 What Is This?

Bulk Image Generator lets you design a visual template once and automatically generate
hundreds or thousands of personalized images from an Excel spreadsheet.

**Perfect for:**
- 🎓 Student ID cards & certificates
- 🏷️ Product labels & price tags
- 📛 Event badges & passes
- 📣 Personalized social media posts
- 🏆 Award certificates & diplomas
- 📦 Bulk e-commerce product images

---

## ✨ Features

- 🖼️ **Visual Template Editor** — drag, drop, and resize placeholder boxes on a canvas
- 📊 **Excel Import** — reads `.xlsx` and `.xls` files with automatic column mapping
- ⚡ **Multi-threaded Generation** — uses all CPU cores via `Parallel.ForEach` for maximum speed
- 🔤 **Text Placeholders** — configurable font, size, color, alignment, and word wrap
- 📷 **Image Placeholders** — supports cover, contain, and stretch fit modes with corner radius
- 💾 **Template Save/Load** — save your designs as `.bigt` files and reload them anytime
- 📁 **Assets Folder** — point to a folder of images; Excel only needs filenames, not full paths
- 🚫 **100% Offline** — zero internet connection required, ever
- 🪟 **No Installation** — single `.exe` file, just double-click and run

## 🚀 Quick Start
Download & Run (End Users)
1. Download `BulkImageGenerator.exe` from [Releases](../../releases)
2. Double-click to run — no installation, no .NET download needed
3. If Windows SmartScreen appears → click **"More info" → "Run anyway"**
